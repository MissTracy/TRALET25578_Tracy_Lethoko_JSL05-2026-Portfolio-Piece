# 📝 Kanban Task Management App

A responsive Kanban task management application built with vanilla JavaScript, HTML, and CSS.

## 🌐 Live App Demo
*https://tracyskanban.netlify.app/ — Netlify*

## 👩🏽‍💻 Project Presenation Link
*https://youtu.be/8FFnvpSWjog*

---

## 📌 Features

- Create, edit, and delete tasks
-  Organize tasks into columns (Todo, Doing, Done)
- Toggle Dark mode / Light mode 
- Fully responsive (desktop, tablet, mobile)
- Mobile sidebar navigation
- Persistent data using local storage (or API if applicable)

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (ES Modules)
- Local Storage / API

---

## 📲 Responsive Design

The app is fully responsive:
- Desktop: Full sidebar view
- Tablet: Adaptive layout
- Mobile:
  - Sidebar becomes a slide-out menu

---

##  UI Features

- Sidebar toggle
- Theme switch (dark/light)
- Dynamic logo switching based on theme
- Modal-based task editing
